# Markdown 格式的语法支持说明
lixun-robot 完全支持标准 Markdown 语法的文档解析，包括标题层级（# 到 ######）、加粗（**内容**）、斜体（*内容*）、列表（- 或 1.）等格式。解析时会自动过滤语法符号，仅保留核心文本内容，但会在分段时参考标题层级，确保同一标题下的内容作为关联片段。

## Markdown 文档的分段逻辑
1. 一级标题（#）和二级标题（##）会作为分段边界，不同标题下的内容拆分为独立片段；
2. 同一标题下的内容，按空行分隔为多个片段，每个片段不超过250字；
3. 列表项（- 或 1. 开头）会被整体保留，不会拆分单个列表项，确保语义完整。

例如，本 Markdown 文档中的“Markdown 文档的分段逻辑”下的3个列表项，会被合并为一个完整片段，检索时若用户提问涉及“分段逻辑”，会直接返回整个列表内容。此外，Markdown 中的链接（[链接文本](url)）会被提取为“链接文本（对应链接：url）”的形式，确保链接信息不丢失。
